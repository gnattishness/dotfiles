My dotfile repository

Installation
============

#. Install `homely <https://homely.readthedocs.io/en/latest/>`__ (recommend using `pipx <https://pipxproject.github.io>`__
    .. code-block:: console

        $ pipx install homely

#. Add the repo

    .. code-block:: console

        $ homely add https://github.com/gnattishness/dotfiles.git
